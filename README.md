# MLB Swing Grades

Interactive 2026 MLB swing-grade leaderboard powered by Driveline Baseball's proprietary
swing-path models. 20-80 scouting scale, 50 = cohort mean, ±10 = 1 σ. Minimum 100
qualifying swings.

**Live:** https://samehrlich.github.io/mlb-swing-grades/

Data sourced from ClickHouse regular-season MLB pitch-by-pitch. Grades are computed
from each batter's current swing metrics and rescaled to the league cohort.
